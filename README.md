# PrepPilot AI – Mock Interview Simulator

AI-powered tool for students to practice job interviews.  
Get real-time feedback on your answers and a final summary report.

## ✨ Features
- Choose from 4 roles: Software Engineer, Data Scientist, Product Manager, Marketing Specialist
- Dynamic AI‑generated interview questions
- Instant feedback on clarity, relevance, and completeness
- Final summary with strengths, improvements, and actionable tips
- Transcript review

## 🛠️ Tech Stack
- Python + Streamlit (frontend & backend)
- Google Gemini 1.5 Flash (LLM)
- Deployed on Streamlit Community Cloud
