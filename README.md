# PrepPilot AI – Mock Interview Simulator

An AI-powered mock interview tool that helps students practice job interviews with real-time feedback and personalized summaries.

## ✨ Features
- **7 Interview Roles** (Frontend, Backend, Data Scientist, ML Engineer, DevOps, Product Manager, HR)
- **Dynamic AI-generated questions** (Gemini API or rule-based fallback)
- **Instant feedback** with rating (0-10) and improvement suggestions
- **Light/Dark theme** toggle
- **Progress tracking** during interview
- **Final report** with strengths, improvements, and downloadable transcript
- **Export interview** as .txt file

## 🛠️ Tech Stack
- Python 3.10+
- Streamlit (UI framework)
- Google Gemini API (AI) or built-in rule-based fallback
- Custom CSS for styling
